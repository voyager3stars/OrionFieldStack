# LogMigrator 1.6

LogMigrator は、OrionFieldStack の撮影・解析データマスターログである `shutter_log.json` の構造を、古いフォーマット（v1.5.0 など）から最新の **v1.6.0** 仕様へと安全に変換（マイグレーション）するための単独ユーティリティツールです。

## 機能・特徴

- **自動バックアップと安全設計**
  処理を実行する前に、同ディレクトリへ自動的に `.bak` のバックアップファイルを作成します。万が一パースエラーなどが発生した場合は、元の内容を即座に復元し、ファイル破損を防止します。
- **データ構造のクリーンアップ**
  旧バージョンでルート直下や `location` に散在していた各種メタデータ（`timezone` や `utc_offset`）などを、新仕様で定義された階層へ整理・削除します。
- **解析値 (analysisツリー) の再編**
  `analysis` ブロック直下にあった SkySolveEngine (SSE) 関連の変数や、一括でまとめられていた StarFlux 関連の数値類を、新しい `SSE` および `SF` オブジェクトへ独立・分離し、未実行項目にはデフォルト値である `"pending"` を適切に割り当てます。
- **重複実行防止**
  既に `"version": "1.6.0"` としてタグ付けされているレコードを検出した場合は、自動的に処理をスキップするため、何度実行しても安全です（冪等性の確保）。

## 使い方

ターミナルから、変換を行いたい `shutter_log.json` のファイルパスを引数として渡し、スクリプトを実行してください。

```bash
python logmigrator_1_6.py /path/to/your/shutter_log.json
```

処理が完了すると、自動的にJSON構造がv1.6.0へアップデートされます。
※当プログラムは JSON の変換のみに特化しており、旧フォーマットのCSVに対する処理は行いません。
